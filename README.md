## 🚀 Setup


1. Clone the repository
2. Open `Grocery.xcodeproj` in Xcode
3. Select your target device/simulator
4. Press `⌘ + R` to build and run

## 🛠️ Tech Stack

- **Language:** Swift
- **UI Framework:** SwiftUI
- **Persistence:** SwiftData
- **Architecture:** MVVM
